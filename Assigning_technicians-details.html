Details

the TRADIVTEL project for managing network site installations. The project aims to develop a web app that enables efficient communication between management and field technicians. TRADIVTEL, a telecom company, faces challenges in coordinating installation projects due to communication gaps and delays in meeting deadlines. To overcome this, a web app will be implemented to provide real-time project tracking, task assignments, resource management, and accurate reporting. Issue: TRADIVTEL's network site installation projects encounter communication and deadline tracking problems. Communication between management and technicians is slow and error-prone, while meeting project deadlines is hindered by poor tracking. Objectives: TRADIVTEL's project seeks to enhance communication by implementing a real-time platform for management and technicians. Additionally, a real-time deadline tracking system will be established to improve project timeline management and prevent delays.
